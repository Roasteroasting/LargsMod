# LargsMod

The Largs Mod for Terraria, WIP

This mod will add the "Largs" class to Terraria, which is planned to contain, but will not be limited to, the following features:



* 20 - 40 new weapons
* Mobile and Static weapon subtypes
* 5 - 15 armor sets
* 10 - 20 accessories
* Crafting materials



Currently this mod is very early in development and very little actual content has been implemented; that which *has* been implemented is mostly for testing, with some pretty scuffed sprites and things. 99% of what is currently in this repo is highly subject to change.
